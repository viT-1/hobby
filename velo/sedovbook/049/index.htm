Во время езды по равнине гонщику нужно избегать резких замедлений движения и ускорений. Целесообразнее идти по дистанции более равномерно, так как даже в высоком темпе при равномерном педалировании организму спортсмена будет легче справляться с нагрузкой, чем при неравномерном движении. Резкие ускорения, рывки и другие элементы неизбежно влекут за собой замедление темпа движения. Гонщик делает в среднем 100—120 оборотов в минуту. Техника педалирования во время установившегося режима движения по равнине зависит от скорости движения и поставленной передачи (рис. 22).

Большое значение для гонщика имеет умение выбирать правильные передачи и вовремя их применять, так как скорость, развиваемая велосипедистом, может изменяться в больших пределах. Во время езды по ветру при укладке 8,50 м велосипедист за 100 оборотов проходит дистанцию 850 м, во время движения против сильного ветра при укладке 4,95 м всего 495 м, т. е. почти вдвое меньше (рис. 23).

Зная временные характеристики и конкретные усилия, которые применяет гонщик, идущий с определенной соревновательной скоростью по равнине, можно подбирать для него специальные упражнения, которые были бы близки по характеру нагрузок к соревновательным. Эти упражнения должны выполняться с частотой движений, приближающейся к частоте педалирования при заданной скорости.

Использование таких упражнений даже в соревновательном периоде скажется благотворно на работоспособности гонщика. Регистрация усилий во время педалирования при помощи аппаратуры может оказать помощь тренеру в процессе самой тренировки. Зная скорость, с какой двигается гонщик, и время, которое он способен идти с заданной скоростью, можно найти оптимальный для данного спортсмена отрезок дистанции, который он может свободно в данное время проходить с максимальной для себя скоростью. Проводя подобные наблюдения над группой велосипедистов, можно подобрать команду из гонщиков, находящихся примерно на одном уровне тренированности.
